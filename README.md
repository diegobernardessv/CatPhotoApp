# Cat Photo App

This is an interactive web page project dedicated to cat photos, demonstrating fundamental HTML5 structures and CSS3 styling.

## Features

*   **Image Gallery**: Display of cat photos with external links.
*   **Informative Lists**: Sections detailing things cats love and hate, using ordered and unordered lists.
*   **Submission Form**:
    *   Selection of environment type (Indoor/Outdoor) via radio buttons.
    *   Selection of cat personality via checkboxes.
    *   Text field for submitting photo URLs with required field validation.
*   **Responsive Design**: Layout adaptable for mobile and desktop devices.

## Technologies Used

*   HTML5
*   CSS3 (Internal styling)

## How to Run

To view the project, simply open the `index.html` file in any modern web browser.
